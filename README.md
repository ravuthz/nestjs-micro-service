# NestJS Micro Service ( TCP )


## Description
NestJS micro service with default protocol ( TCP ) and PostgreSQL database.

## Installation

```bash
yarn install
# or
pnpm install
```

## Running the app

```bash
# development
yarn start
# or
pnpm start

# watch mode
yarn start:dev
# or
pnpm start:dev

# production mode
yarn start:prod
# or
pnpm run start:prod
```

## Test

```bash
# unit tests
yarn test
# or
pnpm test

# e2e tests
yarn test:e2e
# or
pnpm test:e2e

# test coverage
yarn test:cov
# or
pnpm test:cov
```

## Create this project
```bash
npx degit https://github.com/ravuthz/nestjs-micro-service micro_product
```

```bash
git clone npx degit https://github.com/ravuthz/nestjs-micro-service micro_product
```

NOTE:
